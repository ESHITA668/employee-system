# 🧑‍💻 Employee Management System (Python + SQLite + OOP)

## 📌 Project Overview
This is a simple **console-based Employee Management System** built using Python, SQLite, and Object-Oriented Programming (OOP).  
It allows users to perform CRUD operations like adding, viewing, searching, updating, and deleting employee records.

---

## 🚀 Features
- ➕ Add new employee
- 📄 View all employees
- 🔍 Search employee by ID or name
- ✏️ Update employee details
- ❌ Delete employee record
- 💾 SQLite database integration (persistent storage)

---

## 🛠️ Tech Stack
- Python 3
- SQLite3 (Database)
- OOP Concept
- Command Line Interface (CLI)

---
## 📂 Project Structure
Employee-Management-System/
│
├── main.py # Entry point of the application (runs the system)
├── database.py # Handles SQLite database connection and table creation
├── employee.py # Employee class (OOP model for employee data)
├── operations.py # CRUD operations (Add, View, Update, Delete logic)
├── utils.py # Helper functions (input validation, formatting, etc.)
├── employees.db # SQLite database file (auto-generated)
└── README.md # Project documentation

---
## 🧠 Design Approach
- Followed modular programming to separate logic into different files
- Used OOP principles to represent employee data as objects
- Used SQLite for lightweight and persistent data storage
- Focused on clean code, modular design, and reusability.
